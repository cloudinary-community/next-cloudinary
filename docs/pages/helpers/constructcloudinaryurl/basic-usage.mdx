# Getting Started with constructCloudinaryUrl

## Basic Usage

The `constructCloudinaryUrl` function is the core abstraction that builds URLs for Next Cloudinary.

It uses the [Cloudinary URL Gen SDK](https://github.com/cloudinary/js-url-gen) with the parsed props to construct and build transformations and configurations for an image.

Basic usage includes an `options` and `config` property:

```
constructCloudinaryUrl({
  options: {
    src: 'turtle',
    width: 100,
    height: 100
  },
  config: {
    cloud: {
      cloudName: '<Cloud Name>'
    }
  }
});

// https://res.cloudinary.com/<Cloud Name>/image/upload/c_limit,w_100/f_auto/q_auto/turtle
```
