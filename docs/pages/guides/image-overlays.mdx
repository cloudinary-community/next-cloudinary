# Overlaying Images

You can add images on top of other images by using image-based overlays.

## Example

<div style={{ maxWidth: 500, margin: '0 auto' }}>
  <CldImage
    width="960"
    height="600"
    src={`${process.env.IMAGES_DIRECTORY}/turtle`}
    sizes="100vw"
    overlays={[{
      publicId: 'images/earth',
      position: {
        x: 50,
        y: 50,
        gravity: 'north_west',
      },
      effects: [
        {
          crop: 'fill',
          gravity: 'auto',
          width: 500,
          height: 500
        }
      ]
    }]}
    alt="Turtle with earth"
  />
</div>
