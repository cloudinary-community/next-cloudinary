# Getting Started with CldVideoPlayer

The CldVideoPlayer component helps to embed Cloudinary videos using the [Cloudinary Video Player](https://cloudinary.com/documentation/cloudinary_video_player) giving you a full customizable experience for your player.

## Basic Usage

To add a video player, use the CldVideoPlayer component with the basic required props
`width`, `height`, and `src` and import the Video Player CSS on the page
that you'll be using the component.

<HeaderImage>
  <CldVideoPlayer
    id="basic"
    width="1620"
    height="1080"
    src={`${process.env.VIDEOS_DIRECTORY}/mountain-stars`}
  />
</HeaderImage>

<CodeBlock>
  ```jsx copy showLineNumbers
  import { CldVideoPlayer } from 'next-cloudinary';
  import 'next-cloudinary/dist/cld-video-player.css';

  <CldVideoPlayer
    width="1920"
    height="1080"
    src="<Public ID>"
  />
  ```
</CodeBlock>

<Callout emoji={false}>
  Using the App Router in Next.js 13+? Add the `"use client"` directive at the top of your file.
</Callout>

## Customization

You can further take advantage of features like customizing your player:

<HeaderImage>
  <CldVideoPlayer
    id="customization"
    width="1620"
    height="1080"
    src={`${process.env.VIDEOS_DIRECTORY}/mountain-stars`}
    colors={{
      accent: '#ff0000',
      base: '#00ff00',
      text: '#0000ff'
    }}
    fontFace="Source Serif Pro"
  />
</HeaderImage>

<CodeBlock>
  ```jsx copy showLineNumbers

  import 'next-cloudinary/dist/cld-video-player.css';

  <CldVideoPlayer
    width="1620"
    height="1080"

      base: '#00ff00',
      text: '#0000ff'
    }}

  />
  ```
</CodeBlock>

## Player Events

Or listening to player events for advanced interactions with:

<HeaderImage>
  <CldVideoPlayer
    id="player-events"
    width="1620"
    height="1080"
    src={`${process.env.VIDEOS_DIRECTORY}/mountain-stars`}
    onMetadataLoad={({ player }) => {
      console.log(`[CldVideoPlayer] Duration: ${player.duration()}`);
    }}
    onPause={({ player }) => {
      console.log(`[CldVideoPlayer] Current Time: ${player.currentTime()}`);
    }}
  />
</HeaderImage>

<CodeBlock>
  ```jsx copy showLineNumbers
  import { CldVideoPlayer } from 'next-cloudinary';
  import 'next-cloudinary/dist/cld-video-player.css';

  <CldVideoPlayer
    width="1620"
    height="1080"
    src="<Cloudinary URL>"
    onMetadataLoad={({ player }) => {
      console.log(`[CldVideoPlayer] Duration: ${player.duration()}`);
    }}
    onPause={({ player }) => {
      console.log(`[CldVideoPlayer] Current Time: ${player.currentTime()}`);
    }}
  />
  ```
</CodeBlock>

<Callout emoji={false} type="info">
  See your browser's web console to preview the information logged in the above callbacks.
</Callout>
